# VK-бот (CHEREPOVETS | ОТЧЁТЫ)

Эта папка — полный проект VK-бота. Заливается в ОТДЕЛЬНЫЙ репозиторий/проект Vercel бота.
Сайт лежит в корне проекта и к этой папке отношения не имеет.

## Что куда заливать

| Что | Куда |
|---|---|
| Всё содержимое этой папки (vk-bot/) | Репозиторий бота -> Vercel проекта бота |
| google-apps-script/send-form-response-to-vk.gs | Google Таблица -> Расширения -> Apps Script (заменить весь код) |
| supabase/*.sql | Уже применены, трогать не нужно |

## Что нового в этой версии

api/vk.js:
- Команда /беседа создать <тип> (staff, candidates, reports, ai, general) — бот сам создаёт беседу с полным доступом и присылает ссылку-приглашение
- Команда /неактив <с> <по> <причина> — заявка на отпуск из ЛС, уходит на сайт в раздел «Неактивы» + алерт в STAFF
- Команда /дайджест — сводка команды за 7 дней (отчёты, одобрения, топ-5 по XP)
- Автопроверка дисциплины при «/состав синхронизировать»: при 2/2 предупреждениях или 3/3 выговорах бот шлёт алерт руководству

google-apps-script/send-form-response-to-vk.gs (v49):
- При повышении строка модератора переносится в секцию новой должности (в свободный слот или новую строку), старый слот очищается под вакансию
